12 September 2002.

The alpha codebase is about to undergo a significant change in accordance with recent discussions on the Octal-dev mailing list. The project has moved toward a MIDI-esque sequencing and channel model, having abandoned the strict step-sequencing and numeric hex notation of tracker programs. In addition, large parts of Octal will be moved to GNU Objc, a powerful object-oriented language that's part of the GNU Compiler Collection.

Watch this space for new information.

About the project

OCTAL is a GNU project (started in 1999) to create a free music system based on unit generator synthesis, sampling, and audio-plugin technology. It consists of:
  1. A music-systems plugin API, used to define sound processing objects called "machines" that users may download from the web to use as instruments or effects in their compositions.
  2. A core component to load these machines, communicate with them, route and mix audio signals between them, and produce output on an audio device.
  3. A graphical user interface (using the Gimp Toolkit) to interactively create, compose, and perform music using the core as a backend.
  4. An open file format for saving these compositions and the associated resources (sound samples, plugin machines) that comprise them.

As of March 2001, part one is essentially stable; parts two and three are currently in usable "developer's releases" and are progressing quickly. The eventual goal of this project is to create a completely free-software music system (plugin modules and all) and an archival format so that the work put into compositions can last with time.
